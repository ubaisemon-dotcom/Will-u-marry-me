# Marriage Proposal Webpage

This webpage allows users to create a personalized marriage proposal based on the names provided via URL parameters. Below is a guide on how to use the webpage effectively.

## How to Use
1. **Access the Webpage**: Open your browser and navigate to the marriage proposal webpage.

2. **Customize Names**: You can customize the names that appear on the proposal by adding parameters to the URL. Simply append `?name1=YourName&name2=PartnerName` to the URL.
   - For example: `https://example.com/marriage-proposal?name1=John&name2=Jane`
   - This will display the proposal addressing John and Jane.

3. **View the Proposal**: After customizing the names in the URL, hit enter, and the proposal will reflect the changes.

## Example URL
- To propose to Alice, use: `https://example.com/marriage-proposal?name1=Alice&name2=YourName`

Feel free to experiment with different names to create the perfect proposal!
